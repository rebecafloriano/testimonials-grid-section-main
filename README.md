# Frontend Mentor - Testimonials Grid Section Solution

![Project Preview](./preview.jpg)

This is a solution to the [Testimonials grid section challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/testimonials-grid-section-Nnw6J7J7y). This project focused on building a complex grid layout that is fully responsive and visually accurate to the design.

> **Live Demo:** [rebecafloriano.github.io/testimonials-grid-section-main/](https://rebecafloriano.github.io/testimonials-grid-section-main/)

## 🚀 The Challenge

The objective was to build a testimonials grid section and get it looking as close to the design as possible. Users should be able to view the optimal layout depending on their device's screen size.

### My focus during this challenge:
* Implementing a complex layout using **CSS Grid**.
* Handling different card colors and background decorations (like the quotation icon).
* Ensuring a smooth transition between mobile and desktop views using Media Queries.

## 🛠️ Built With

* **Semantic HTML5** markup.
* **CSS Custom Properties** (Variables) for easy color management.
* **CSS Grid** for the main mosaic layout.
* **Flexbox** for internal card alignment.
* **Mobile-first workflow**.

## 📂 Project Highlights

* **Grid Areas:** Used `grid-template-areas` to define the layout structure clearly in the CSS.
* **Typography:** Integrated Google Fonts as specified in the style guide.
* **Assets:** Managed profile images and background SVG icons to match the original design.
